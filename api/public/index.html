Search & Specs
GET /search?query=:q Search devices ▼

Searches GSMArena for devices matching the query. Returns a list of matching results with slugs.

Query Parameters
query string · required Device name to search for. e.g. samsung galaxy s25
Example
/search?query=pixel 9 pro ⎘ copy
GET /:slug Device specs by slug ▼

Returns full specifications for a device by its GSMArena slug. Get the slug from /search results.

Path Parameters
slug string GSMArena device slug e.g. samsung_galaxy_s25_ultra-12559
Example
/samsung_galaxy_s25_ultra-12559 ⎘ copy
GET /phone?name=:name ★ MAIN ▼

One endpoint for everything. Pass a plain device name — it searches, fetches specs, and scrapes camera samples all in one shot. Best starting point for any integration.

Query Parameters
name string · required Plain device name. e.g. samsung galaxy s25 ultra, iphone 16 pro max
Response includes
brand, model Device identity fields
specifications object Full specs table (chipset, display, battery, connectivity…)
device_images string[] Official press images from the specs page
hdImageUrl string High-res hero image from review page (1200px)
cameraSamples array All camera sample tabs — Main, Night, Zoom, Selfie, Video with classified images
lensDetails array Per-lens metadata from review page
review_url string GSMArena review page URL if available
Example
/phone?name=samsung galaxy s25 ultra ⎘ copy
Reviews & Camera Samples
GET /review/:reviewSlug Full review data ▼

Scrapes a GSMArena review page and all camera-sample tabs. Returns hero images, article images grouped by section, and all camera sample categories.

Path Parameters
reviewSlug string Full review slug e.g. samsung_galaxy_s25_ultra-review-2939p5 or without page suffix samsung_galaxy_s25_ultra-review-2939
Response includes
heroImages string[] Header / top-of-page images
articleImages object In-body images grouped by nearest section heading
cameraSamples array All tabs (Main Camera, Night, Zoom, Selfie, Video…) each with classified images
Example
/review/samsung_galaxy_s25_ultra-review-2939p5 ⎘ copy
GET /review/:reviewSlug/camera-samples Camera samples only ▼

Returns only the camera samples section for a review — all tabs with classified images. Lighter response than the full review endpoint.

Path Parameters
reviewSlug string GSMArena review slug
Example
/review/samsung_galaxy_s25_ultra-review-2939p5/camera-samples ⎘ copy
GET /review/:reviewSlug/images Hero + article images ▼

Returns only the hero and article images for a review page — excludes camera samples. Useful for fetching high-res editorial images.

Path Parameters
reviewSlug string GSMArena review slug
Example
/review/samsung_galaxy_s25_ultra-review-2939p5/images ⎘ copy
